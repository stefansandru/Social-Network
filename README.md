# Social Network App
This is a Java-based application with a JavaFX graphical interface that simulates a social network.
It allows users to manage profiles, friendships, and interactions efficiently.
The application connects to a PostgreSQL database for storing user and friendship data.

## Features
After you log in with an existent user in the data base, you can
- add and remove friends,
- view pending friend requests,
- send messages to selected users,
- and search for users by name.

## Getting Started:
To run the application, you need Java JDK,
PostgreSQL with the required database schema, and the JavaFX library configured in your IDE.
Configure the PostgreSQL connection URL, username, and password in the application’s settings before launching the program.
